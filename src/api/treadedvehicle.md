# TreadedVehicle

TreadedVehicle is a Vehicle with treads, such as a tank or a tonk.

## Properties

| Property Name | Return Type | Description | Tags |
| -------- | ----------- | ----------- | ---- |
| `turnSpeed` | `number` | Returns the turn speed in degrees per second. | Read-Only |
