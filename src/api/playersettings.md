# PlayerSettings

## Description

Settings that can be applied to a Player.

## API

### Functions

| Function Name | Return Type | Description | Tags |
| -------- | ----------- | ----------- | ---- |
| `ApplyToPlayer(Player)` | `None` | Apply settings from this settings object to Player. | Server-Only |
