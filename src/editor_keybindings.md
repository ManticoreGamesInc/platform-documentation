| Main Menu                           | Esc                                        | Ctrl + P                 |           | Opens Pause Menu                                                                      | None                            |
| **Hierarchy Organization**          |                                            |                          |           |                                                                                       |                                 |

| Create New Folder                   | N                                          | Ctrl                     |           | Create A New Folder In The Hierarchy                                                  | None                            |
